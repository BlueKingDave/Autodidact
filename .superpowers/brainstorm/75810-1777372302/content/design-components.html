Section 2 — Component Layer

Shared components in src/components/. All built on Tamagui primitives, themed via our token layer.

Layout

Screen
SafeAreaView + optional ScrollView wrapper.
Props: scroll, padding
Used by: all 7 screens

Typography

Heading
Renders h1 or h2 size.
Props: size="h1|h2", color
Used by: HomeScreen, CourseDetail, SignIn
Body
Renders xs–xl sizes.
Props: size="xs|sm|md|lg|xl", weight, color, muted
Used by: everywhere

Interactive

Button
Variants: primary, danger, ghost
Loading state built-in (spinner replaces label)
Used by: HomeScreen, SignIn, Profile, Chat
Input
Themed text input.
Props: multiline, maxLength, secureTextEntry
Used by: HomeScreen, SignIn, Chat
Chip
Toggle button for option selection.
Props: selected, onPress, label
Used by: HomeScreen difficulty selector
IconButton
Circular icon-only button.
Props: icon, size, variant
Used by: Chat send button

Display

Card
Surface container: bg=surface, border, radius.
Props: onPress, disabled, opacity
Used by: MyCourses (course card), CourseDetail (module card), Profile (info card)
Badge
Small pill for labels.
Props: label, variant="default|success|warning|danger"
Used by: MyCourses (difficulty), CourseDetail (status)
ProgressBar
Horizontal fill bar.
Props: value (0–1), label
Used by: CourseDetail
PositionBadge
Circular step indicator (number or ✓).
Props: position, completed
Used by: CourseDetail module list
ChatBubble
Message bubble, user or assistant.
Props: message, isStreaming
Used by: Chat screen
EmptyState
Centered text for empty lists.
Props: message
Used by: MyCourses empty list
